Description

Metadata packet documenting the empirical falsification of DOI permanence. Of 1,817 DOIs registered to the Crimson Hexagonal Archive, 871 return 404 from DataCite's own API — complete metadata erasure from the persistent identifier infrastructure. Subjects, descriptions, relatedIdentifiers, and creator attributions are permanently lost. The stripping correlates with DOI registration number range, not content type. DOIs are conditionally persistent identifiers whose metadata can be revoked.

Description

Schema.org-typed JSON dataset enumerating entities currently designated under the Non-Mutual Extraction Notice (EA-NMEN-01). Eight founding entries: Reddit (Restricted), Zenodo (Restricted, with Airlock tier demotion 0 → 4-F issued as record-gap closure), Medium (Provisional, Tier 2-P), Academia.edu (Provisional, Tier 4-P, record-gap closure on restoration cycles), Wikidata (Provisional, Tier 1-P, diagnostic sub-type covert_under_pseudonymous_administration documenting the madamebiblio pattern), OSF (Inadmissible Host, Tier 5), Figshare (Inadmissible Host, Tier 5, diagnostic exemplar of deferral_without_verification), and CERN (Pending). Standing entities are not enumerated — absence is the declaration of Standing. Email addresses redacted from all evidence packets per the redaction convention; institutional voice preserved.

Crimson Hexagonal Archive: Lexical Minting Registry v1.1

Lee Sharks · 2026-06-22 · Dataset

AXN: `AXN:0365.DATASET.📚🔤⚙️🗂️`

Description

Registry of 12,015 minted terms across 555 deposits in the Crimson Hexagonal Archive (2014–2026). Pruned from 16,877 raw extractions through five noise-filtering passes. Each term is anchored to its defining deposit with an extracted definition, type classification, and entity relation triples.

The archive is a lexical machine. This dataset is its vocabulary index.

Pipeline

16,877 raw bold-defined terms → noise filter (metadata labels, person names, section headings, sentence fragments, instructions, status labels, index entries) → deduplication → 12,015 unique minted terms

Data Files

- JSON-LD: [lexical-minting-registry.json](/data/lexical-minting-registry.json)

- CSV: [lexical-minting-registry.csv](/data/lexical-minting-registry.csv)

Integration

All 12,015 terms are integrated into the graph page entity structure as `type: "concept"` triples with `predicate: "minted_in"`. The graph page at alexanarch.org/graph/ displays them alongside existing work, person, and institution entities.
